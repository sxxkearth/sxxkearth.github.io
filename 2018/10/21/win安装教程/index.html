win系统安装
发表于 2018-10-21| 更新于 2023-06-14|系统安装
字数总计: 1.5k|阅读时长: 5 分钟| 阅读量:

本文是一篇电脑win系统安装的教程，我会尽可能争取详细，本文涉及的安装方法将会包含，win7及以上的系统安装，安装方式包括MBR+Legacy BOIS及GPT+UEFI引导，包括双系统新添加引导及修复引导的方法。同样，也会涉及iso及gho镜像的安装，格式不一样，安装方式自然也不尽相同。接下来，就开始吧。

安装方式的选择

更换新的系统，有很多很多的方式。常见的包括：1、通过各种一键重装软件协助安装。 2、其实，直接解压系统镜像，双击运行里面的Setup.exe也可以直接运行安装程序。3、制作直接安装的系统盘。即，选择U盘启动后，直接进入win语言选择页面。4、通过PE工具。等等。

这么多方式，该怎么选呢，嗯再有u盘的情况下，首选必然是PE下安装，主要是因为，系统安装，总是有可能会有些意外的，没有u盘的情况下，才不得不选用第1，2，种方法，但是1，2种的前提条件是电脑本身有系统，且能正常运行的前提之下的。PE安装，虽然不能避免风险，但是却给解决问题提供了一个很好的基础。本人使用的PE为IT天空出品的优启通PE制作工具。PE制作，一键就好啦！

引导方式的选择

系统的引导分为两种，一种是MBR+Leagacy BOIS。这种方式主要是win7 及win7以下的系统例如xp，基本上都是属于这种引导方式安装的。
另一种是GPT+UEFI引导安装的，是相对前者来说的新型的引导方式，个人觉得它的优点主要是引导文件归列清晰，多系统之间文件不会过多相互影响，并且支持快速启动。相对来说，我用同样的方式安装win10，UEFI的开机速度，普遍都会比前者快那么一点点。

引导方式的选择，也基本是毋庸置疑的，能使用UEFI的情况，最好是使用UEFI安装。

系统版本 可选的引导方式
win xp MBR
win7 32 MBR
win7 64 MBR/UEFI
win8 MBR/UEFI
win8.1 MBR/UEFI
win10 MBR/UEFI

基本的设置要求

BOSI设置

本人现有一台联想G510以及一台华硕A455l，主板的设置，将主要以这两台电脑这两种主板进行介绍。网传大部分的设置，都是设置BOIS下为U盘第一启动，嗯，在此，我个人不是这么首选的，一直习惯了联想的还原键，选择boot menu项，就是可启动的列表了。设置之前，关闭Security–Secure Boot menu–Secure Boot Control 选选项为Disable。 联想通过还原按键/F12可进入Boot menu选项，华硕通过ESC进入Boot menu选项。

Secure Boot Control Disable

硬盘，分区设置

系统安装，引导方式的选择不是根据主板的设置而变化的。而是根据硬盘的状态自动选择的。即：如下图，若为上面的HD0硬盘分区形式，带efi分区，即安装系统默认为UEFI引导，若为下面的HD1分区形式，即：扩展分区+盘，即默认为MBR+Legacy BOIS引导。

要确认是哪种分区形式，使用DG分区工具，在硬盘HD0位置右击，若转换为MBR格式灰显，则现在为MBR分区，若转换为GUID格式灰显，则现在为GPT分区形式。 默认MBR分区只能有一个主分区，即C盘，剩下的分区为扩展分区。默认GPT分区格式，为多个主分区，基本全部为主分区。GPT分区格式是，需要新建一个EFI分区，大小约为200 ~ 500 M，我给的是500M。

详细的分区可参照下图设置。

HDO为GPT格式，HD1为MBR格式

系统安装

在准备工作做完之后，我们就可以开始准备系统盘的东西了。开头写的四种安装方式中，前1，2，种过于简单，本文不予介绍。下面为第三四种安装方法的操作。ISO文件下载地址：MSDN,GHO本人用的比较少，不做推荐。PS：ISO为原版镜像，GHO为别人修改过的。

win安装盘制作，安装。

适合ISO，这种方式会自建引导，但win7UEFI可能会报错。用UltreISO工具，打开ISO文件之后，启动–写入硬盘映像–硬盘驱动器选要制作的U盘–写入方式USB-HDD+ –写入。然后 等待即可。

进入Boot menu，选择制作的U盘。进入设置语言步骤，然后，一直下一步操作就好了。系统版本如专业版，家庭版的选择在后面输入相应的序列号即可。

PE下安装

PE下ISO文件的安装

PE安装ISO，先使用PE自带的虚拟光驱，对镜像文件进行挂载，然后使用win安装器安装。第一栏选择光驱路径下/sources/install.wim文件。第二栏，UEFI选择EFI分区，MBR选择c盘，不能错选，否则会报错。第三栏选择你想要的C盘的位置。然后，选择开始安装即可。

PE下GHO文件的安装

GHO文件使用ghost安装器安装/OneKey Ghost恢复工具。恢复工具操作简单，选择还原分区，镜像路径，c盘路径，确定即可。

引导修复

mbr引导修复

老pe下的引导修复工具即可。选择自动修复

UEFI引导修复

新机型WIN10 PE下的uefi修复工具。操作也很简单。

其他工具

在此推荐两种手动添加启动项的工具，适合自定义。

MBR下 EasyBCD 提取码：u3yl

UEFI下 EasyUEFI 提取码：c58k
